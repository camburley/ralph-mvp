# Demo goal

Run `npm test` in this project. If a test fails, read the source file in `src/`, fix the bug in the implementation (not the test), and re-run `npm test` until every test passes.

When all tests are green, set `"done": true`.

Constraints:
- Do NOT modify any `*.test.ts` file.
- Only edit files under `src/`.
- Keep the public API identical: `export const add = (a: number, b: number) => number`.
