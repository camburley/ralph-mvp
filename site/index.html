ralph-mvp

Terminal-based agentic coding loop. ~550 LOC total, ~150 in the loop core. No chat UI. No humans in the middle. The agent plans, executes, observes, and self-corrects against real terminal feedback — in a strict loop — until the goal is done.

Install & run the demo

git clone https://github.com/camburley/ralph-mvp
cd ralph-mvp
npm install
cd demo && npm install && cd ..
npx tsx src/index.ts run demo/goal.md --cwd demo

Ships a broken vitest. Agent fixes it autonomously and commits — while you watch.

The loop

1. Context Loading  ←  MCP filesystem server (real MCP)
2. Plan             ←  claude -p (swappable brain)
3. Execute          ←  allowlisted shell / write_file
4. Observe          ←  stdout / stderr / exitCode captured
5. Self-correct     ←  observation folds into next prompt
        │
        └──→ loop until done=true

Why it matches the brief

  1. Terminal-based, not a chat UI — `commander` CLI, streaming terminal output.
  2. Ralph-style autonomous loop — 5 explicit phases, strict state machine in `loop.ts`.
  3. Real MCP integration — spawns `@modelcontextprotocol/server-filesystem` over stdio, uses the MCP SDK client.
  4. Self-correction on terminal output — every observation (stdout/stderr/exitCode) feeds back into the next prompt.
  5. Swappable brain — `brain.ts` is ~40 LOC around `claude -p`; swap to any function-calling LLM.
  6. Resumable — `.ralph/state.json` ledger, `ralph resume` picks up mid-run.
